DFS

I ) Installation du rôle Réplication DFS

On accède au menu Gérer > Ajouter des rôles et fonctionnalités.

Dans la section Avant de commencer cliquer sur Suivant >;

Dans la section Sélectionner le type d’installation, sélectionner Installation basée sur un rôle ou une fonctionnalité;

Dans la section Sélectionner le serveur de destination, sélectionner le serveur puis cliquer sur Suivant >;

Dans le section Sélectionner des rôles de serveurs, sélectionner ==Réplication DFS== puis cliquer sur Suivant >;

Accéder au menu Gestion du sytème de fichier distribué DFS

Cliquer sur Réplication > Cliquer Nouveau groupe de réplication > Sélectionner Groupe de réplication mutli-usage

Renseigner un Nom et le domaine (auto-rempli normalement)

⚠  Il faut avoir installer le rôle Réplication DFS sur tout les serveurs concernés par le transfère`

Ajouter ensuite les serveurs concerné pour la réplication

Sélectionner MAile Pleine :

Sélectionner Répliquer en continu à l'aide de la bande passante spcécifiée

Sélectionner le membre Principal et cliquer

Puis cliquer sur Ajouter pour sélectionner les dossiers a répliquer : [DFSRoots = Nom sans importance]

Cliquer sur Suivant une fois que le dossier est choisi.

Choisir ensuite le chemin de destination où les dossiers seront répliqué [DFSRoots = Nom sans importance]

Cliquer ensuite sur suivant puis Créer
