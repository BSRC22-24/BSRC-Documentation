AD DS Core

Ajouter le rôle AD DS:

Add-WindowsFeature -Name AD-Domain-Services –IncludeManagementTools

Créer un domaine "as.local"

Install-ADDSForest -DatabasePath C:\Windows\NTDS -DomainMode 7 -DomainName AS.local -
DomainNetbiosName AS -ForestMode 7 -InstallDns:$false -LogPath C:\Windows\NTDS -
NoRebootOnCompletion:$true -SysvolPath C:\Windows\SYSVOL -Force:$true

Ajouter un sous domaine au domaine "as.local"

Install-ADDSDomain -ParentDomainName "as.local" -Credential (Get-Credential) -InstallDns:$false -SkipPreChecks

Il faudra saisir par la suite :

  • Saisir le nom du sous domaine (uniquement le sous domaine)
  • Puis définir un mot de passe
