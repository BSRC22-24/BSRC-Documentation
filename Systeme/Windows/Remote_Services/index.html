Services Bureau à distance

I ) Installation du rôle Services Bureau à Distance

On accède au menu Gérer > Ajouter des rôles et fonctionnalités.

Dans la section Avant de commencer cliquer sur Suivant >;

Dans la section Sélectionner le type d’installation, sélectionner Installation basée sur un rôle ou une fonctionnalité;

Dans la section Sélectionner le serveur de destination, sélectionner le serveur puis cliquer sur Suivant >;

Dans le section Sélectionner des rôles de serveurs, sélectionner Services Bureau à distance puis cliquer sur Suivant >;

Dans la section Sélectionner des fonctionnalités cliquer sur Suivant >

Dans la section Services Bureau à distance, cliquer sur Suivant >;

Dans la section sélectionner des services de rôle, sélectionner Accès Bureau a distance par web, Passerelle des services Bureau à distance, Hôte de session Bureau a distance et Service Broker pour les connexions Bureau à distance puis cliquer sur Suivant >;

Dans la Section Rôle Web Server (IIS), cliquer sur Suivant >;

Dans la section sélectionner des services de rôle, cliquer sur Suivant >;

Dans la section Services de stratégie et d’accès réseau, cliquer sur Suivant >;

Dans la section Confirmer les sélections d’installation, cliquer sur Installer.

II) Installation des Services Bureau à distance

On accède au menu Gérer > Ajouter des rôles et fonctionnalités.

Dans la section Avant de commencer cliquer sur Suivant >;

Dans la section Sélectionner le type d’installation, sélectionner Installation des services Bureau à distance;

Dans la section Sélectionner le type de déploiement, sélectionner Démarrage rapide

Dans la section Sélectionner le scénario de déploiement, sélectionner Déploiement de bureaux basés sur une session;

Dans la section Sélectionner un serveur, cliquer sur Suivant >

Dans la section Confirmer les sélections, cocher Redémarrer automatiquement le serveur de destination si nécessaire et cliquez sur Déployer.

Une fois le service installer, on peut cliquer sur le lien en bas de la fenêtre pour atteindre la page de connexion au RemoteAPP

⚠  Un redémarrage du serveur peut être nécessaire.
