Configuration Réseau

Ajouter Sudo

apt update

apt install sudo

adduser ["Nom d'utilisateur"] ["Nom de Groupe"]

🔎  sudo !! reprend la commande précédente avec élévation de privilège

  • Afficher les configuration réseaux :

nano /etc/network/interfaces

  • Configuré le réseaux :

``` # Statique # auto [nom int] iface [Nom int] inet(=ipv4) static address 192.168.1.100 # Your desired static IP address netmask 255.255.255.0 # Your subnet mask gateway 192.168.1.1 # Your gateway (router) IP address dns-nameservers 8.8.8.8 # DNS server(s) - You can add multiple servers

# DHCP # auto [Nom int] iface [Nom int] inet dhcp ```

  • Redémarrer le réseaux :

systemctl restart networking

ipconfig= ip a
