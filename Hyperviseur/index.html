Hyperviseur

ESXi

[!important]

Il y a toujours un interface réservé a l’hyperviseur.

Pour pouvoir avoir plusieurs réseau sur le même serveur, il faut créer des Vswitch, une fois créé on doit attribuer ces Switch virtuel a des ports en créer des Groupes de port.

VSwitch

Accéder a la catégorie Mise en réseaux dans le menu de gauche, Dans le sous menu Commutateur virtuel cliquez sur Ajouter un Commutateur virtuel :

  • Donner nom a ce commutateur et choisissez la liaison (Interface Physique; noté active si un câble est connecté )
  • Dans le sous menu Groupe de Port, cliquez sur ajouter un groupe de port. Donnez lui un nom et attribuez lui un **commutateur virtuel **

Hyper-V

Ouvrir le Gestionnaire Hyper-V via le menu Outils.

Proxmox

Pour chaque réseaux, il faut 1 port RJ45. Il faut 1 port supplementaire pour la machine hyperviseur. Pour 2 machines virtuelles sur reseau diff, il faut donc 3 ports RJ45 (2 pour les VM + 1 hyperviseur)

Dans le menu Server View, derouler le menu Datacenter, et cliquer sur [==NomDuServeur==]

VSwitch :

Accéder au menu Network en déroulant le menu System.

Créer un nouveau VSwitch en cliquant sur Create > Linux Bridge.

Remplir les informations du formulaire avec le nom du Bridge, l’IP, son masque en notation CIDR et le port Physique de la machine (format : enp1s0)

[!CAUTION]

Il ne peut y avoir qu’une seule passerelle pour tout les Bridges (laisser la première d’origine)

Créer une VM :

Cliquer en haut a droite sur Create VM et remplissez les informations demandés.

Dans le le menu Network, verifiez bien le Bridge, Laisser le modelpar defaut.
